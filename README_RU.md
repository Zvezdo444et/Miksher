# 🎧 Miksher - Java-аудиомикшер и менеджер устройств

Miksher — это профессиональное настольное приложение, разработанное на Java, предназначенное для использования нескольких устройств вывода для совместного просмотра или прослушивания. Вы можете подключить несколько устройств (динамики/наушники) и регулировать громкость и использовать каждое устройство индивидуально.

## ✨ Возможности

* **Управление воспроизведением аудио:** Используйте основной модуль `AudioPlayer` для точного управления потоковой передачей и воспроизведением аудио.

* **Управление устройствами:** Управляйте и отслеживайте аудиоустройства с помощью `DeviceManager`.

* **Интуитивно понятный пользовательский интерфейс:** Создан с использованием выделенного слоя пользовательского интерфейса (`MainFrame`, `DeviceControlPanel`) для удобства использования.

* **Модульная архитектура:** Следует чистому шаблону, подобному MVC, разделяющему основную логику, пользовательский интерфейс и бизнес-логику/управление устройствами.

## 🚀 Начало работы

### Предварительные условия

На вашей системе должен быть установлен Java Development Kit (JDK).

### Инструкции по сборке
Для сборки проекта перейдите в корневой каталог репозитория и выполните:

```
# Предполагается использование Maven, на основе pom.xml
mvn clean install

Вы также можете создать установщик, обернув файл .jar с помощью Launch4j и создав установщик с использованием компилятора Inno Setup (пример кода включен в проект)

```
